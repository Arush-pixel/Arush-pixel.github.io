# Personal Website

This project is a personal website created to showcase my interests, projects, and aspirations. It is designed to provide visitors with an overview of who I am, my passion for robotics and technology, and my active involvement in sports.

## Project Structure

The project consists of the following files:

- **index.html**: The main entry point of the website, featuring a brief introduction and links to other pages.
- **about.html**: A detailed page about myself, including my interests in robotics, technology, sports, and my career goals in the movie industry.
- **projects.html**: A showcase of my robotics projects, including descriptions and links to the work I have done.
- **css/styles.css**: The stylesheet that sets the background color to blue and styles the layout and fonts, including airplane graphics.
- **js/scripts.js**: JavaScript file for any interactivity on the website, such as navigation and animations.

## Getting Started

To run this website locally, follow these steps:

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser.
3. Navigate through the website using the provided links.
